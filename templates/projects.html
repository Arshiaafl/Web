APItweet

Technologies Used: Streamlit, Langchain, OllamaLLM, Newspaper, Python

Movie Recommendation App

Developed a movie recommendation system using Python and Streamlit, leveraging collaborative filtering techniques to provide personalized movie suggestions.

Responsibilities:

  • Data Loading: Extracted and loaded movie and rating datasets from a zip file, preprocessing the movie titles for consistency.
  • User-Item Matrix: Created a user-item matrix from the ratings data to facilitate similarity calculations between movies.
  • Item Similarity Calculation: Implemented cosine similarity to determine how similar movies are to one another based on user ratings.
  • Recommendation Engine: Developed a recommendation engine to suggest movies based on the similarity scores of a given movie.
  • Streamlit Interface: Designed an interactive UI with Streamlit to allow users to input a movie name and receive personalized recommendations.

Technologies Used: Streamlit, pandas, scikit-learn, Python, zipfile

Outcome: Successfully built a functional movie recommendation application that provides users with relevant movie suggestions based on collaborative filtering techniques, enhancing the user experience with a simple and intuitive interface.

Text Summarization

Developed a FastAPI-based service for summarizing articles using pre-trained language models.

Responsibilities:

  • API Development: Created an API endpoint using FastAPI to handle POST requests for article summarization.
  • Data Processing: Integrated the Hugging Face Transformers pipeline to perform text summarization.
  • Model Integration: Utilized a pre-trained summarization model to generate concise summaries of input text.
  • Response Handling: Implemented logic to return summaries in a user-friendly format, ensuring accurate and meaningful responses.

Technologies Used: FastAPI, Pydantic, Transformers, Python

Outcome: Successfully built a responsive API for summarizing text articles, allowing users to obtain concise summaries with ease, and demonstrating effective integration of NLP models into web services.
